# Comment-Extractor

## Status: DATA COLLECTION (80%)
**Priority**: MEDIUM
**Category**: Support Tools

## Description
Playwright-based web scraping tool to extract comments from client social media pages. Stores data in SQLite.

## Tech Stack
- **Scraping**: Playwright
- **Database**: SQLite
- **Backend**: Python/Node.js

## What's Working
- Social media comment extraction
- Multiple platform support
- SQLite storage
- Batch processing
- Playwright automation

## What's Missing

- Scheduling system
- Dashboard for results
- Export functionality
- Rate limiting
